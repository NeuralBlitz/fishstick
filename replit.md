# Project Overview
This project is an LLM demo/research environment with both Python and Node.js components. It was imported from GitHub and configured to run on Replit.

## Recent Changes
- Configured Node.js frontend workflow on port 5000 with `index.js` as the entry point.
- Installed Python and Node.js dependencies.
- Added `express` as a dependency for the frontend server.
- Created `index.js` to provide a basic landing page.

## Project Architecture
- **Frontend**: Node.js/Express server (port 5000).
- **Backend/Logic**: Python-based LLM tools and experiments.
- **Dependencies**: Managed via `package.json` and `requirements.txt`.
