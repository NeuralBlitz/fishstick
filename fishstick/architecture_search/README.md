# Architecture Search

Neural architecture search methods.

## Installation

```bash
pip install fishstick[architecture_search]
```

## Overview

The `architecture_search` module provides tools for automated neural architecture search.

## Components

| Component | Description |
|-----------|-------------|
| `Supernet` | Supernet for efficient NAS |
| `Controller` | Neural architecture controller |
| `SearchSpace` | Search space definitions |
