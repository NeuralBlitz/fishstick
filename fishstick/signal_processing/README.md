# Signal Processing

Signal processing utilities and transforms.

## Installation

```bash
pip install fishstick[signal_processing]
```

## Overview

The `signal_processing` module provides signal processing utilities including filter banks, time-frequency analysis, and wavelet transforms.

## Components

| Component | Description |
|-----------|-------------|
| `FilterBanks` | Filter bank implementations |
| `TimeFrequency` | Time-frequency analysis |
| `FourierOperators` | Fourier transform operators |
| `WaveletTransform` | Wavelet transforms |
