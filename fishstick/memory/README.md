# Memory Mechanisms

Memory-augmented neural networks and differentiable data structures.

## Installation

```bash
pip install fishstick[memory]
```

## Overview

The `memory` module provides memory mechanisms including Neural Turing Machines and differentiable stacks.

## Components

| Component | Description |
|-----------|-------------|
| `NTM` | Neural Turing Machine |
| `MemoryAttention` | Memory attention mechanisms |
| `ContentAddressing` | Content-based memory addressing |
| `DifferentiableStack` | Differentiable stack |
